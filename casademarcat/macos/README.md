# Casa de marcat MacOS
Plugin-ul Oblio casa de marcat compatibil cu sistemul de operare MacOS.

## Descarcare
https://obliosoftware.github.io/builds/casademarcat/macos/oblio-casademarcat.arm64.dmg

## Instalare
Deschide in Finder folderul "Downloads" si drag and drop la "Oblio Casa de Marcat.app" in sectiunea "Applications"

## Limitari
Momentan functioneaza doar cu casele Datecs
